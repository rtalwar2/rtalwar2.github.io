ROBOT FIGHT (EVERY SUMMER)

This all started with one of my best friends who challenged me for a robot fight. We both had to make a robot by the end of summer and compete in a competition.

The rules are simple:

‚Ä¢  All types of electronics are allowed.
‚Ä¢  We must design our own weapons and can 3D print them. 
For my robot I used a servo motor and a stepper motor as actuators. My controller was a simple joystick and an Arduino Uno microcontroller. Now the most important part, the weapons, for my weapons I used items I could find at home. Eventually I ended up using sparkle sticks firework as my main weapon. TripAdvisor - ML project

This was a Machine Learning project completed as part of the Machine Learning course at Ghent University. The project aimed to explore and make predictions on a dataset of 2500 TripAdvisor listings in East-Flanders, scraped from the website. The data consisted mostly of tabular, text, and image data.

The project consisted of three sprints:

Exploratory Data Analysis (EDA): Conducted a thorough EDA of the data, including finding interesting insights such as recommended restaurants for dog-walking and the location of expensive restaurants in Ghent.

Traditional ML Methods: Built predictive models using traditional ML methods, including a recommendation system based on image clustering, a "budget filter" model, and an "influencer maker" model to determine measures for maximizing restaurant reviews.

Deep Learning Methods: Improved the recommendation system from sprint 2 with the use of deep learning methods, including anomaly detection to improve classification models and text generation models such as fine-tuning GPT-2 to generate restaurant reviews.

MASTER THESIS (09/2022 - 06/2023)

Exploring Explainable AI techniques for Detecting Contamination Features for improved Machine Learning-based Intrusion Detection. The goal of this thesis is to develop a methodology that detects contaminating features in intrusion detection system (IDS) datasets using explainable artificial intelligence (XAI) techniques such as SHAP. Additionally, the inter-dataset generalization technique is employed to assess the impact of these features on the generalization ability of machine learning (ML) models.

BACHELOR THESIS WITH AUTOMOTIVE DRONES (02/2022 - 06/2022)

My bachelor thesis is in collaboration with Jan De Nul. Our task is to use drones to perform autonomous 
object location measurements. 
For their project, positioning items are critical to perform their operations according to the design/plan. 
The positioning of for example buoys is done by support vessels at the moment. Our task is to automate 
part of this cumbersome process using semi-autonomous drones.

We made an Android application in android studio and used the DJI mobileSDK to communicate with 
the drone. In this project we used a variety of sensors in the drone and some Machine learning for object detection to realize the georeferencing of buoys through a video stream.

MICROSERVICE APPLICATION (09/2022 - 12/2022)

For my Systems Design course, I had the opportunity to work on a project that involved building a microservice-based application using the Quarkus framework. The objective of the project was to design and implement a system that could handle a high volume of requests and handle them efficiently.

To achieve this goal, me and my team decided to use the Kafka message queue for asynchronous communication between the services and REST for synchronous communication. This allowed us to decouple the services and ensure that each service could handle its own workload without being affected by the performance of other services.

We also used a NoSQL database for each service, which allowed us to scale the database easily and handle large amounts of data. 

Throughout the project, I learned a lot about systems design and building a microservice application. I gained experience in working with the Quarkus framework, Kafka, and REST, as well as NoSQL databases.

Overall, this project was a great learning experience and helped me to understand the importance of designing a system that can handle a high volume of requests and how to achieve that by using microservices and message queues.
